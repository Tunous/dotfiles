Tunous's dotfiles
=================

Installation
------------

```sh
# Clone repository
git clone https://github.com/Tunous/dotfiles.git ~/dotfiles
cd ~/dotfiles

# Use Makefile to symlink configs
make install
```

Used software
-------------

* Bspwm      - Window manager
* Compton    - Compositor
* Dmenu      - Dynamic menu
* Dzen2      - Bar/popups
* Mutt       - Mail User Agent
* Ranger     - File manager
* Sxhkd      - Hotkey daemon
* Tmux       - Terminal multiplexer
* Vim        - Text editor
* Vimperator - Vim key bindings for Firefox
* Zsh        - Shell
