Dotfiles
========

My personal configuration files for Linux-based systems.

Installation
------------

To install these dotfiles simplly clone this repository and run installation script:

```
$ git clone https://github.com/Tunous/dotfiles.git ~/.dotfiles
$ ~/.dotfiles/install
```
